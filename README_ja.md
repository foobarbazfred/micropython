# MicroPythonリポジトリ

このリポジトリには、ハードウェアコンポーネントタイプ別に整理されたMicroPythonドライバ、ライブラリ、サンプルプログラムが含まれています。

## ディレクトリ構成

### 📷 [camera](./camera)
**Arducamカメラドライバとサンプルプログラム**

Arducamカメラモジュール用のドライバと動作確認用のサンプルコードが含まれています。このディレクトリには、ハードウェアインターフェース実装と、MicroPythonに対応したマイコンボード上でのカメラ機能デモンストレーションプログラムが含まれています。

---

### 📺 [displays](./displays)
**ディスプレイドライバライブラリと例**

様々なディスプレイモジュール（LCD、OLED、電子ペーパーなど）のドライバとテストプログラムが含まれています。ディスプレイ使用方法を示すサンプルプログラムが含まれています。

---

### 📡 [mqtt-security-patterns](./mqtt-security-patterns)
**セキュリティ考慮を含むMQTTプロトコル実装**

MicroPython用のMQTTクライアントライブラリと安全な通信パターンが含まれています。暗号化接続、認証方法の例、およびセキュリティレベルに応じたMQTT通信のコーディング例を置いています。

---

### 🌡️ [sensors](./sensors)
**マルチセンサドライバとテストプログラム**

以下を含む包括的なセンサドライバコレクション：
- **AT42QT1011** - タッチセンサ
- **BME280** - 環境センサ（気圧、温度、湿度）
- **BME680** - ガスセンサ付き環境センサ
- **FSR402** - 圧力センサ
- **GP2Y0A21YK** - 赤外線距離センサ
- **HC-SR04** - 超音波距離センサ
- **LSM9DS1** - 9軸IMUセンサ
- **MAX30100** - パルスオキシメータセンサ
- **MAX31855** - 熱電対アンプ
- **S13683-03DT** - フォトダイオード
- **VL53L1X** - ToF距離センサ

各センササブディレクトリには、ドライバコードとサンプルアプリケーションが含まれています。

## 使い始める

1. ハードウェアのニーズに基づいて、関連するコンポーネントフォルダに移動します
2. 各サブディレクトリのドライバドキュメントを確認します
3. サンプルプログラムとテストファイルをリファレンス実装として使用します
4. 特定のMicroPythonボードとアプリケーションに合わせてコードを調整します
